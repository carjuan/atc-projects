What is Flexbox?

Flexbox gives you the ability to arrange items in ROW or COLUMN. Flex items grow or expand IF there is additional space.

Let's Play With Flexbox

Without FLEXBOX, elements will go in block, one under the other.

The size of the container will always be the size of the content. In this case, the size of each box inside the container.

Let's use FLEXBOX to FLEX the items inside the container to be one NEXT to the other.

What if we want to stack elements on top of each other, like in a COLUMN?

flex-direction: column will help you achieve that

NOTE: Notice that without flexbox, items inside the container OVERFLOW because the height of the contents is greater than the container's

<- that is called OVERFLOW

When there is available space in the container, we can start MOVING elements around!

Open the devTools and modify: justify-content and align-items properties of the container element.

NOTE: with no height set, by default, elements stretch in the available space

What if you want all your children (FLEX ITEMS) elements to have the same width in the available space of the container?

We use the GROW property on the children elements to let them expand equally.
